== Description ==

Reference guide for all admin UI elements in WordPress admin. Must-have for all plugins developers.

* Titles & icons
* Block Widget, layout
* Draggable block widgets
* Tables
* Pagers
* Tabs
* Forms
* Buttons, links, images
* Content sliders
* Lightbox windows

For now not all of theese are described, but I'm working hard on this.

== Installation ==

1. Upload `wordpress-admin-ui-reference-guide` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
